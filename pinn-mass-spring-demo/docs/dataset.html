How We Build the Dataset

To keep the experiment fully reproducible we generate synthetic data from the differential equation itself. We simulate the mass from 0 to 10 seconds and record 1,000 evenly spaced time points. These dense samples form our “ground truth” reference curve.

Real experiments are never perfect, so we also create noisy observations by adding gentle Gaussian noise with a standard deviation that is only 2% of the signal amplitude. This mimics sensors that are mostly accurate but still wobble a little.

During training we pretend that we only measured a small subset of time points (for example the first 40 points between 0 and 5 seconds). This tiny dataset is what we hand to the neural networks.

See the first notebook (Mass Spring Physics and Data) for the full code that creates these curves step by step.

Synthetic data keeps the tutorial safe to run anywhere while still feeling realistic.
