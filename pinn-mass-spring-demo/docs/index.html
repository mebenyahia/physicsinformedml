PINN Demo – Mass Spring System

This small site accompanies the repository and provides a guided tour of the physics, data, and neural networks used in the demo. Everything is written for curious beginners: we explain new ideas slowly, link back to the notebooks, and highlight the most important observations.

A Physics Informed Neural Network (PINN) is a neural network that learns from measurements and from the differential equation describing the system. By respecting the physics, the model stays stable even when we have little or noisy data.

Use the navigation bar to explore the equation that governs the mass and spring, how we generate synthetic data, what models we train, and how the results compare.

Getting started

Want to run the code yourself? Visit the project README for setup instructions. Once your environment is ready, open the notebooks in the notebooks/ folder to follow the experiments step by step.
