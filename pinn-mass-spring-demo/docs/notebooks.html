Explore the Jupyter Notebooks

The heart of this project lives in two interactive notebooks. They contain the code, plots, and friendly explanations. You can view them directly on GitHub or download the repository to run them locally.

  1. 01_mass_spring_physics_and_data.ipynb – builds the physics intuition and creates the dataset.
  2. 02_pure_nn_vs_pinn.ipynb – trains the baseline neural network and the PINN, then compares them.

If you would like a quick read-only view, run the notebooks once and export them to HTML (File → Download as → HTML). Place the exported files into this folder and link them here for easy browsing.

The notebooks are fully annotated so you can learn by running the code yourself.
